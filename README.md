# KIE Kogito IntelliJ Plugin

An open-source IntelliJ plugin that brings **Apache KIE Kogito** BPMN and DMN editors to IntelliJ IDEA.  
This project extends the Kogito tooling ecosystem by enabling developers to **create, edit, validate, and visualize** business process and decision models directly within IntelliJ.

---

## 🧭 Overview

The **KIE Kogito IntelliJ Plugin** provides full integration with the [Apache KIE Kogito](https://kogito.kie.org) runtime and modeling tools.  
It allows developers to model:
- **BPMN 2.0** business process diagrams  
- **DMN 1.3** decision tables and decision requirements diagrams (DRDs)  
- (Future support planned) **PMML** predictive models  

This plugin is built on top of the **KIE Tooling** framework that powers the official [VS Code Kogito Bundle](https://github.com/apache/incubator-kie-tools).

---

## 🚀 Features

- 🧩 **Native BPMN and DMN editors** integrated directly into IntelliJ IDEA  
- 🧠 **Kogito-compatible models** — ready for use in Kogito or Drools runtimes  
- 🪶 **Lightweight and offline-capable** — no external services required  
- ⚙️ **Validation and syntax checking** for BPMN and DMN models  
- 🔄 **Seamless round-trip editing** between IntelliJ and VS Code Kogito editors  
- 🌐 **Open-source foundation** built on the Apache KIE ecosystem
